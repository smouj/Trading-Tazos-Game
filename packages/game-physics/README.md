# @ttg/game-physics

Battle physics engine for Trading Tazos Game.

## What's here
- Slam simulation (`simulateSlam`)
- Collision detection & resolution
- Flip mechanics
- Impact scoring (`scoreImpact`, `scoreBettingImpact`)
- AI slam generation (`generateAISlam`)

## Dependencies
- `@ttg/game-core` — types and constants
